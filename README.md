Titanic Survival Prediction

Objective:
To predict whether a passenger survived the Titanic disaster using Machine Learning.

Dataset:
Titanic Dataset

Algorithm:
Random Forest Classifier

Libraries Used:
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

Accuracy:
82.12%

Conclusion:
The Random Forest model successfully predicted passenger survival with an accuracy of 82.12%.
